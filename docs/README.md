# NowDoer Landing Page

Welcome to the source code for the **NowDoer** landing page. This project is a high-performance, visually striking web application built with React and Tailwind CSS, designed to convert visitors into "doers".

## Overview

NowDoer is a brand focusing on action over planning. The website reflects this with a brutalist, high-contrast aesthetic, strong typography, and smooth animations that emphasize momentum.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1.  Clone the repository.
2.  Install dependencies (assuming a Vite environment):
    ```bash
    npm install
    ```
    *Required libraries:* `react`, `react-dom`, `framer-motion`, `lucide-react`, `tailwindcss`.

### Running Development Server

```bash
npm run dev
```

### Building for Production

```bash
npm run build
```

## Key Features

-   **Modern Aesthetics:** Dark mode default with neon red accents.
-   **Responsive Design:** Mobile-first architecture ensuring perfect rendering on all devices.
-   **Animations:** Powered by `framer-motion` for scroll reveals and micro-interactions.
-   **Component Architecture:** Modular React components for maintainability.
